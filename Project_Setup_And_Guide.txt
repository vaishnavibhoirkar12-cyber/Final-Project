Hello! If you are a beginner, this guide will explain exactly what this software does and how you can run it perfectly.

-----------------------
1. WHAT WE BUILT
-----------------------
We built a web platform where Faculty can post assignments and Students can upload their projects (PDFs). Normally, a teacher has to read every PDF to grade it. Instead, we built a custom "Keyword Auto-Grader"!

When a student uploads their PDF, our system instantly reads all the text inside the PDF and mathematically checks it against a list of required keywords that the Faculty member provided. It then gives an instant percentage grade. For example, if a teacher requires 4 specific topics, and the student's essay only mentions 3 of them, the system instantly grades it a 75%.

-----------------------
2. HOW TO RUN THE APP
-----------------------
Step 1: Make sure you have Python installed on your computer.
Step 2: Open your terminal (Command Prompt or VS Code Terminal).
Step 3: Make sure you are inside the project folder.
Step 4: Install the required packages by typing this exactly:
        pip install flask flask-sqlalchemy flask-bcrypt pymupdf
Step 5: Start the server by typing:
        python app.py
Step 6: Open your web browser (Chrome, Edge, etc.) and go to: http://127.0.0.1:5000/
